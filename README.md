# bt3-skill-shader-editor
A command-line modding tool that edits the **shader files (DAT &amp; V00)** of **Blast 1's and Blast 2's** in DBZ Budokai Tenkaichi 3. A tribute to [Maycon](https://www.youtube.com/@dbzmadeiradabr303), who'd written his version of the tool in Python last year.
